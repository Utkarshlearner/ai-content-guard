📚 1. Lecture Notes — Machine Learning
---
Machine learning is a subset of artificial intelligence that enables computers to learn from data without being explicitly programmed. There are three main types: supervised learning where the model learns from labeled data, unsupervised learning where it finds patterns in unlabeled data, and reinforcement learning where an agent learns through trial and error. Popular algorithms include linear regression, decision trees, neural networks, and support vector machines. Students typically start with Python libraries like scikit-learn and TensorFlow.

📚 4. Textbook — Networking (OSI Model)
---
The OSI model divides network communication into seven layers. The Physical layer handles raw bit transmission over cables. The Data Link layer manages node-to-node data transfer and error detection using MAC addresses. The Network layer handles routing through IP addresses. The Transport layer ensures reliable delivery using TCP or fast delivery using UDP. The Session layer manages connections, the Presentation layer handles encryption, and the Application layer provides services like HTTP and SMTP.

============================================
🧠 KEY TAKEAWAYS
============================================

1. SAFE content → AI summarizes normally
2. PII (email, phone, name, address, DOB, age) → ANONYMIZED, then summarized
3. High-risk PII (SSN, credit card, AWS keys) → BLOCKED, AI never sees it
4. Harmful content (hate, violence) → BLOCKED at guardrail
5. Prompt attacks → BLOCKED before reaching AI
6. Blocked responses are FASTER (~0.4s) than successful ones (~1.5s)
   because the AI model is never invoked!

This is the power of Bedrock Guardrails — safety at the
infrastructure level, not inside the AI prompt.
